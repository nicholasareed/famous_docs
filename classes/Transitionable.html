Transitionable Class

Defined in: transitions/Transitionable.js:14

A state maintainer for a smooth transition between numerically-specified states. Example numeric states include floats or Transform objects.

An initial state is set with the constructor or set(startState). A corresponding end state and transition are set with set(endState, transition). Subsequent calls to set(endState, transition) begin at the last state. Calls to get(timestamp) provide the interpolated state along the way.

Note that there is no event loop here - calls to get() are the only way to find state projected to the current (or provided) time and are the only way to trigger callbacks. Usually this kind of object would be part of the render() path of a visible component.
