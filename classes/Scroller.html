Scroller Class

Defined in: views/Scroller.js:10

Scroller lays out a collection of renderables, and will browse through them based on accesed position. Scroller also broadcasts an 'edgeHit' event, with a position property of the location of the edge, when you've hit the 'edges' of it's renderable collection.

Constructor

Scroller

(
  • [options]
  • [clipSize=undefined]
  • [margin=undefined]
)

Defined in views/Scroller.js:10

Parameters:

  • [options] Options optional

    An object of configurable options.

    • [direction=Utility.Direction.Y] Number optional

      Using the direction helper found in the famous Utility module, this option will lay out the Scroller instance's renderables either horizontally (x) or vertically (y). Utility's direction is essentially either zero (X) or one (Y), so feel free to just use integers as well.

  • [clipSize=undefined] Number optional

    The size of the area (in pixels) that Scroller will display content in.

  • [margin=undefined] Number optional

    The size of the area (in pixels) that Scroller will process renderables' associated calculations in.
