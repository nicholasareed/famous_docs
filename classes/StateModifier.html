StateModifier Class

Defined in: modifiers/StateModifier.js:16

A collection of visual changes to be applied to another renderable component, strongly coupled with the state that defines those changes. This collection includes a transform matrix, an opacity constant, a size, an origin specifier. StateModifier objects can be added to any RenderNode or object capable of displaying renderables. The StateModifier's children and descendants are transformed by the amounts specified in the modifier's properties.
